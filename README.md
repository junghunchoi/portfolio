# 프로젝트 이름: junghun's portfolio

## 프로젝트 소개
이 포트폴리오는 스프링 부트와 Vue.js를 활용하여 개발되었습니다. 
게시판에서 가장 범용적으로 많이 사용되는 기능들을 구현하였습니다.
이 프로젝트는 개발자가 자신의 경력과 프로젝트를 소개하기 위한 목적으로 만들어졌습니다.
